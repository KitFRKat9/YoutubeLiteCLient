To correctly install YoutubeLiteClient x86 you have to do this steps:
1)Run requirements.bat, you need it to install vlc.(If you already have it don't need this step)
2)If you want to keep it portable you can leave it right there.
3)If you want to install it click Install.bat, it will do all the work. (DON'T RUN IT WITH ADMIN PRIVILEDGES)



YouTubeLiteClient Lite (64-bit) is a lightweight YouTube client for weak pc's

Features:
Search videos or paste links directly
Shows 10(optimized for 32bit) results per page with navigation (n = next, b = back)
Choose resolution: 360p, 480p, 720p, 1080p
Stop playback anytime by pressing ENTER
Uses VLC for playback and yt-dlp for streaming
Optimized for low CPU/RAM usage
Requirements
Windows 32-bit or 64-bit
VLC installed (preferably 32-bit)
yt-dlp.exe in the same folder
Python not required if using the standalone .exe
Usage
Launch the client (main64.exe or main32.py)
Choose from the menu:
1 → Search video
2 → Paste link
q → Quit

For search: navigate results with n / b and select a video by number
Pick resolution (360, 480, 720, 1080)
Video opens in VLC. Press ENTER to stop and return to the menu
